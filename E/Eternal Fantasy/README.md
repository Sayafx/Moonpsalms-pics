# 游戏简介

魔族封印大戦勝利100週年慶的尾聲，唯一有王位繼承權利的公主的舉辦成人儀式，這是王國歷史以來最熱鬧的時刻。

元月當空，民眾們為了一睹女王的風采紛紛湧向中央廣場。
在皇家馬車伴隨著響亮的馬蹄聲、民眾的歡呼祝福聲到來的時候，滿月被『災厄の黒竜』所覆蓋。

象徵絕望的雪白的『冬花』在廣場中飛舞、突如其來的巨大魔物、慶典霎時間被哀嚎所掩蓋。
馬車如紙箱般被踩碎，民眾們爭相逃竄。



就連王國兵刃也不放在眼裡的魔物面前，卻有一個稚氣未脫的金發少女站在那裡。

身為警衛團的少年劍士的你好不容易才把少女從魔物手中救出。

然而，在壓倒性的漆黑之牙的攻擊下，就在力量用盡的那瞬間，少女的金發隨風而動，嬌小的身體放出炫目的光輝。

白光中，你聽到了溫柔的聲音並發現等身長的聖劍已經握在手中。在聖劍的引導下你們將魔物擊退。



但是局面已經失去控制，你被誣陷為殺害王女的兇手，不得不離開王都踏上了旅途。



在坎坷的旅途中，你邂逅各種各樣的美女，時而歡笑、時而爭吵，結下羈絆。

然而，在大家一起努力逐一揭開諸多謎團後，面對的卻是絕望和真實……
