在本篇中，玩家所拥有的时间是「四天」，

玩家要利用上午（午前）和下午（午后）两次行动时段，

在你闲逛逡寻的同时，有可能在意想不到的地方遇到意想不到的人。

安定的日常生活不断重复，以及悄悄地进行中的另一个圣杯战争。

命运的布幕将再次升起，那就是四日轮回的规则。

在那前方，有的是虚无，亦或是平稳？
