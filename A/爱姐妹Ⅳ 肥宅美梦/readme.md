# 故事简介：



挣扎于社会的最底层、天涯孤独又不受欢迎的我・西山健二，

终于在夏日的某一天，突然被上司告知我被解雇了。

于是，28岁，无业。

于是，带着稍稍有些自暴自弃的心情，到经常光顾的茶餐厅去吃喜欢的土耳其饭……偶然间听到隔壁座位的对话，

我若无其事地竖起了耳朵。

声音的主人是：有着yin靡肉体的美女医生・藤村江利子和她的丈夫・专业主夫藤村佳祐。

（主夫么……无需工作，便有早晚三餐。哼，何等令人欣羡……）

不管将来如何，我想这一生都与这样的名流夫妇都不可能有所交集呢。

没想到我竟然在同一天内又一次看见了佳祐的身影。

他在我常去的风俗店前，兴致勃勃地往看板和店内窥探着。

如果能好好地利用丈夫去风俗店这一把柄的话……那不就有机会和那个人妻来一发了么？

这一离奇的想法一闪，在我的脑海中浮现出来……

接着数天后……

人妻・江利子漂亮地陷入了我所设下的圈套里，我成功地在Love Hotel里和她度过了浓稠的一晚。

从Love Hotel出来后，板着脸的江利子打的回去了，而我便走路回去……

「你和妈妈干了什么吗？」
