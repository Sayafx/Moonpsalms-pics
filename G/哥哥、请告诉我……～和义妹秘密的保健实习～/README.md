# 游戏简介



现在想起，那是发生在昨天的事。



「哥哥你把自己的身体当教材，教我保健体育！」



在籍名门护理学园的妹妹・百合。

表面上看她的护士之路似乎一帆风顺，

然而其实似乎「一接触到男性的身体的话就会头脑短路」。



不过，把我的身体当教材来实习……是在搞啥？

兄妹做那种事可以吗？就算你说我们并没有血缘关系……



在还没理清思绪时妹妹已经回来了。

但她好像忘了昨天的约定。

悄悄地松了口气，果然还是不需要教保健体育什么的。

然而才刚放心……



「那、那个……哥哥。今晚……保健体育的授课，来做哦……」



顺着对话我不由得点了点头。

今晚，到底会变得怎样呢。一想到这个，心脏的鼓动愈发加速……
