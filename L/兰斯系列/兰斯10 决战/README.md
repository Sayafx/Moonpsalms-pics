# 游戏简介

魔物大军突然开始入侵人类界。
而人类各国却无法顺利达成协作，陷入劣势。

“既然如此，老子就来指挥所有人。”

兰斯的荒唐发言却得到了各国首脑的认同。
而这些首脑，正是兰斯在此前历险中解救，据为己有的女人们。

就这样，兰斯靠床上功夫成功地统一了人类世界。
人类vs魔物，这场最大也是最后的战争就此拉开帷幕。



大陆被拆分为人类与魔物分别居住的两大地区以来，已过去千余年。

这一天，魔物组成的庞大军团突然开始入侵人类界。
魔物界中的势力斗争持续已久，而魔人凯布利斯终于大权在握。
魔物的矛头，就这样转向了人类。

然而，即使面对如此危机，人类国家间的矛盾却仍然未见消解，
统一起来的魔军与人类之间的力量差距已是一目了然。
在一片混乱的首脑会议场上，看不下去的兰斯站了出来。

“你们所有人都来当老子的部下，
只要这样，区区魔军，凭老子就能打他们个屁滚尿流。”

会场自是一片哗然。但喧闹很快便重归平静。
那是因为……

赫尔曼共和国总统：席拉·赫尔曼、

自由都市联盟代表：科潘彤·多铎、

东部岛国JAPAN国君：织田香、

世界最大宗教——AL教法王：库鲁库·莫夫斯。

这些各国家势力的首脑纷纷同意了兰斯的意见。

同时，这些首脑们，也是兰斯在此前的历险之中拯救世界危机，据为己有的女人们。

全人类在兰斯的麾下聚为了一体。
一场大战，即将开始。
