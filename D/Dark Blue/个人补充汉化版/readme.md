# 运行`sfa16_patch.exe`即可



由于原版的补丁有较多问题，我只是将其进行了一定的修复，让其处于一个可以正常阅读的水平



我的日语水平不算太低也不能算高，但已经尽力了



学过一定程度的日语基础语法，通过VNR基本可以阅读所有生肉









但看懂和如何选择合适的词作为合适的翻译是两回事



所以对于一些叫床的句子和单纯的拟声词就不翻译了



还有人物名称的问题，原版翻译有时保持アニス的假名



有时又翻译成爱妮丝，显然它们自己都没有达成统一



我们就直接保留原来的片假算了







还有就是有些句子在喊人名却没写成汉字，像ゆきと，是主角雪斗



因为有时我判断没有翻译的必要，读者要稍微注意下，要认得出来



最后是有些基本的常用句，像谢谢、对不起、再见、厉害之类的平假要认识
